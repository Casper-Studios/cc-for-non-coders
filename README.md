# Claude Code for Non-Coders

A hands-on workshop for non-technical people to learn how to use Claude Code effectively.

## What is Claude Code?

Claude Code is Anthropic's official command-line tool that brings Claude AI directly into your terminal. It can help you with tasks like:

- Automating repetitive work
- Analyzing and processing files
- Creating documents and reports
- Answering questions about your files and data
- Much more!

## Workshop Overview

This workshop is designed for people with little to no coding experience. You'll learn:

1. **Getting Started** - Installing and setting up Claude Code
2. **Basic Commands** - How to interact with Claude in the terminal
3. **File Operations** - Reading, creating, and modifying files
4. **Practical Examples** - Real-world use cases you can apply immediately

## Prerequisites

- A computer with terminal access (Mac, Linux, or Windows with WSL)
- An Anthropic API key or Claude subscription
- Curiosity and willingness to learn!
